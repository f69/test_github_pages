### Списки

Для разметки неупорядоченных списков можно использовать или `*`, или `-`, или `+`:

- элемент 1
- элемент 2
- элемент ...

Между первыми A и B - неразрывный пробел: A&nbsp;BAB

Непосредственно после текста, к которому нужно сделать сноску, помещаем^[Текст сноски.]
